# OSU „Jovche Teslichkov“ Website - 2024

This repository contains the official website for OSU "Jovche Teslichkov", showcasing key information about the school, its programs, contact details, and other valuable resources for students, teachers, and visitors.

The website is designed to provide a user-friendly and informative experience with easy navigation across sections, ensuring that all important aspects of the school are easily accessible.

**Website Link:** [osujtv.pktriot.net](http://osujtv.pktriot.net)

## Project Structure

- **/assets**: Folder for images, fonts, and videos used across the site.
- **/css**: Stylesheets for the website design and layout.
- **/js**: JavaScript code for interactive elements and functionality.
- **/pages**: HTML pages, including the homepage, about page, contact page, and more.
- **/includes**: Common components such as header and footer that are included across all pages.
- **/data**: Contains structured data (optional) such as JSON files for content or documents.

## Features

- **Responsive Design**: The website is fully responsive and mobile-friendly.
- **User-friendly Navigation**: Easy-to-use navigation menu to quickly find important school information.
- **Secure Link**: The site is hosted on a secure subdomain: [osujtv.pktriot.net](http://osujtv.pktriot.net), and has been configured with HTTPS for secure access.
- **Search Functionality**: Quickly find specific content and resources.
- **Interactive Contact Form**: A contact form for visitors to directly get in touch with the school.
